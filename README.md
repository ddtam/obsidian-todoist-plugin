## What is it?

An **unofficial** plugin for Obsidian that offers the ability to have your Todoist tasks sync bidirectionally with Obsidian. Please note that this is not created by, affiliated with, or supported by Doist.

This is a fork of [jamiebrynes7/obsidian-todoist-plugin](https://github.com/jamiebrynes7/obsidian-todoist-plugin) with additional features, distributed as **Todoist Sync+** for personal use.

## Install

This fork is not in the official Obsidian community plugin store. Install via [BRAT](https://github.com/TfTHacker/obsidian42-brat):

1. Install the **BRAT** plugin from the Obsidian community plugin store.
2. Open BRAT settings and choose "Add Beta plugin".
3. Paste this fork's URL: `https://github.com/ddtam/obsidian-todoist-plugin`
4. Enable **Todoist Sync+** in Settings → Community plugins.
5. Paste your Todoist API token when prompted.

BRAT will track the latest commit on this repo's default branch — use BRAT's "Check for updates" to pull new changes.

## Usage

Refer to the upstream [documentation](https://jamiebrynes7.github.io/obsidian-todoist-plugin/docs/overview) for the core plugin features. Fork-specific behavior (e.g. the `todoist-task` single-task badge block, the `Insert Todoist task reference` command) is documented in the project's commit history and code comments.
